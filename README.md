Supermarket Sales Data Analysis

This project performs data analysis and data visualization on a supermarket sales dataset using Python.
It helps to understand product stock status, purchasing behavior, product quantity, and price distribution through graphical visualizations.

Programming Language

- Python

Project Domain

- Data Science
- Data Analysis
- Data Visualization

Libraries Used

- Pandas (for data analysis)
- Matplotlib (for data visualization)

Project Features

- Load supermarket dataset from a CSV file
- Analyze stock status of products (Full, Low, Empty)
- Analyze purchased and not purchased products
- Analyze product quantity
- Analyze product price distribution
- Generate visual charts for better understanding of the dataset

Dataset

The dataset contains supermarket product information.

Example:

Product,Stock_Status,Purchased,Quantity,Price
Rice,Full,Yes,25,50
Milk,Low,Yes,12,30
Oil,Empty,No,0,120
